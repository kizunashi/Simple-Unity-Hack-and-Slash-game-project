This is my first Unity project I've done around a year ago.
Its unlikely that I will make any changes to it in the future, the purpose of it being here is so I can track my progress with future projects and maybe reuse some of the assets and scripts added here.

In this project I did: 
- a movement system, 
- a simple combat system (basically just a character model with a weapon model attached to it playng animations and effects when provided input. there was also a very rough combo system, that would play different animations depending on the amount of nouse clicks and a special attack that would trigger by pressing the specified button on the keyboard) 
- added animations using Unity Animator,
- added a health system and a health bar (I checked its functionality by creating a simple plane which would deal "damage" when a player steps on it),
- a simple 2d design of the healthbar,
- 2 different locations,
- a system of 2 "portals" that would make the player transition between the two scenes.
- my own textues
- some 3d models where also made by me

There are enemies (simple 3d models of slimes that would just move around a certain area. they were supposed to attack and follow the player, but thats when the project became to crumble, as I encountered multiple difficult errors and problems, and lost the motivation to work on the project further).

Issues:

- the healthbar system was very "clunky", sometimes it works, sometimes it does not, so I constantly had to reload the project until It worked. I set up lots of "checks" that would provide infornation in the projects cmd, but was unable to ssolve the issue.

- Unity Animator causing problems. I constantly encountered errors and bugs caused by unity animator, so after some time, some animations where triggered by code and some were triggered by the tool itself which added up to the overall "messines"

- Jumping. This was very hard to make possible, but after some time it started to work somehow. (I think the abstract wording of the sentence provides enough information about the issue)

- enemies. I was unable to make them work properly. they do follow the player, and move within a certain radius but dont attack or recieve damage.

- using assets from both unity and blender. The second location (scene) i was trying to make had the main structure made in blender, and I added some unity assets on top of it. In result it was very hard to add textures on it.
